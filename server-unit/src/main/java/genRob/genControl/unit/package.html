Hier sind Einheiten enthalten, die die Kommunikation zwischen Anwendungen und ihren Roblets ermöglichen.

Einheiten des Roblet-Servers

Jeder Roblet-Server definiert eigene Ressourcen, die ein Roblet zugreifen und verwenden kann.  Der Zugriff geschieht über Einheiten. 

Zur Kommunikation zwischen einer Anwendung und seinen Roblets im Roblet-Server gibt es die Einheiten {@link genRob.genControl.unit.Remotes} und {@link genRob.genControl.unit.Proxies} für die Arbeit mit fernen Instanzen.
